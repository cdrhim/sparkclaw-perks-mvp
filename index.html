How verification works

  1. Admin adds a portfolio company and approved email domains.
  2. Requester submits a Supabase perk request using a work email.
  3. The app auto-matches the email domain to your portfolio records.
  4. If verified, it can auto-email Supabase and start the fulfillment timeline.
